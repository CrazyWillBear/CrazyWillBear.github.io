AP Computer Science Final Project

For my AP Computer Science Spring Final Project, I was tasked with utilizing objects and classes as we were taught throughout the semester to create a program of my choice. I decided to make a text-based game. The game uses classes like Player, Villain, an interface Character, and a parent class with multiple children Artifact. The UML diagram is below:

Class (or other if specified)

Class Description

Character (interface)

An interface for methods that apply to characters (monsters and players)

Player

Class for the player, functions used to keep track of progress and player stats

Villain

Class for villains, used to keep track of villain stats

Animal

Class for animals, child of Villain

Monster

Class for monsters, child of Villain

Artifact

Class for artifacts, keeps track of name and value

Weapon

Class for weapons, keeps track of weapon stats. Players and Villains have weapons, weapons are artifacts
