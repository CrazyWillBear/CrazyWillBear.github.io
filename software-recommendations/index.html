Software Recommendations

  • Web Browsing

    • Web browsing is crucial to the use of one's computer. Information, data, apps, and other important stuff is accessable only through the internet, and web browsers make this experience much easier.

    • Brave

      • Brave is a free, (mostly) open-source internet browser based in Chrome. It is simple, fast, and gives you free money (Brave Rewards).

  • Text Editors

    • Text editors are crucial to the usage of computers. From simply editing documents, to configuring programs, and coding itself. Text editors can make or break a good system.

    • Neovim

      • Neovim is free, open-source, and an improved version of the already amazing text editor Vim. It has the same plugin support, with a few other good features.

    • Vim

      • Vim is free, open-source, and an improved version of Vi. Vim is slightly less bloated than Neovim, and depending on your needs, can actually be better.

    • Notepad++

      • If you're on Windows, nothing can beat the classic Notepad++. It is simple, easy to use, and extremely versatile.

  • Terminal

    • Terminals are crucial at least for Unix systems, as they are what help you interact with your computer on a deep level. Terminal emulators are used for speed and looks.

    • kitty

      • Kitty is free and open-source. It is super fast and lightweight, and allows you to customize the fonts and colors with extreme ease. It's my terminal emulator of choice.

    • Alacritty

      • Alacritty is certainly a fan favorite, and is also free and open-source. However, Alacritty has trouble with customizing fonts and as a result is not my editor of choice. The large community however makes choosing a theme really easy and fun.

  • Operating System

    • Operating systems are the basis for any good computer. In order to use all the software above, you first need an operating system.

    • Artix Linux

      • Artix Linux is my personal favorite Linux distro. It is a version of Arch Linux (my previous favorite). It is customizeable beyond belief, fast, lightweight, and simple. However, it has one advantage over Arch: it doesn't require the use of systemd (I actually use runit).

    • Arch Linux

      • Arch Linux is one of the simplest and most "classic" Linux distros out there. It's been around forever, is super customizeable, fast, lightweigt, and simple. The only issue is that it uses systemd, which is considered by modern standards to be slow and bloated.

    • Pop!_OS

      • Pop!_OS is a beginner Linux distro based in Ubuntu. I vastly prefer it to Ubuntu simply because it has no spyware (unlike Ubuntu) and in my experience has been easier to use and more customizeable. It has all the advantages of using Linux, while being just as easy to use as MacOS or Windows.

  • Shells

    • Shells are how you interact with applications and explore your computer. A high quality shell can make the difference between a shitty and amazing system.

    • zsh

      • zsh is my shell of choice because it is super fast and for a few quality-of-life reasons. First of all, its autocomplete is much faster and functions like a drop down menu. zsh also follows POSIX standards more closely than Bash (which is considered the standard for shells), as well as being super configurable (check out oh-my-zsh).

    • Bash

      • Bash is the most widely used shell in Linux and comes installed by default on many Unix systems. It gets the job done, however with alternatives available, it isn't necessarily my first choice.

    • fish

      • Fish is a pretty great shell, being that it's fast, colorful, has an awesome autocomplete system, and is in general pretty easy to use and install. However, it does not follow POSIX standards pretty much at all, and because of this you cannot run shell scripts.
